# 📸 Ariangram (آرین گرام)

**Ariangram** is a highly creative, Instagram-inspired portfolio showcase application. Instead of standard Instagram Reels, this application features a specialized **"Projects Reel"** where developers can scroll through active projects, making it a unique, interactive resume.

Designed with a sleek, mobile-first approach and capped at a modern `550px` layout for the ultimate app-like experience.

## ✨ Key Features

* **Project Reels (Vertical Scroll):** A custom portfolio feed showcasing active projects with their titles, publishers, short descriptions, view counts, and applicant statistics.
* **Dynamic Social Interactions:** Fully functional interactive heart/like button that toggles states and colors instantly.
* **Interactive Modals:** Open up custom comment sheets and share panels featuring mocked active users.
* **Seamless Navigation:** Complete routing across 6 main views:
    * 🏠 Home Feed
    * 🔍 Explore
    * ➕ Add Post (UI Mockup)
    * 🎥 Projects (Reels Section)
    * 👤 User Profile
    * 🔑 Login Screen
* **Strict Mobile-First Design:** Optimized specifically with a `max-width: 550px` container to ensure a premium mobile simulation on desktops.
